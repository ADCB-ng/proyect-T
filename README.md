#Proyecto-T

Descripcion:

Proyecto grupal de instituto enfocado en el proceso ETL (Extract, Transform, Load) para datos de ventas de una empresa de telecomunicaciones.

Mi enfoque principal fue la Creación de Modelos Gráficos y Consultas Analíticas para representar métricas de ventas y el desempeño segmentado por rangos.

Uso de tecnologias: 
Python
Analisis de procesamientos con libreria Pandas 
Vizualizacion de graficos con la libreria Matplotlib y Seaborn
Flujo de implementacion de proceso ETL
